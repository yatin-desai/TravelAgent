# CRM System

A simple CRM system to manage customer accounts and track the sales cycle (leads, opportunities, contracts).

## Tech Stack
- Frontend: React
- Backend: Node.js/Express
- Database: PostgreSQL

## Features
- Customer account management
- Sales cycle tracking: leads, opportunities, contracts
- Authentication
- RESTful API

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`
